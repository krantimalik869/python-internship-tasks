## Task 1 - Hangman Game

**Goal:** A text-based Hangman game where the player guesses a word one letter at a time.

**Features:**
- 5 predefined words selected randomly
- Maximum 6 incorrect guesses allowed
- ASCII hangman drawing that updates with each wrong guess
- Tracks correct and incorrect letters
- Play again option at the end

**Key Concepts:** `random`, `while loop`, `if-else`, `strings`, `lists`

## Task 3 - Task Automation

**Goal:** Automate small real-life repetitive tasks using Python scripts.

**Features:**
- **3A:** Moves all `.jpg` files from one folder to another using `shutil`
- **3B:** Extracts all email addresses from a `.txt` file using `regex` and saves them
- **3C:** Scrapes the title of a webpage using `requests` and saves it to a file

**Key Concepts:** `os`, `shutil`, `re`, `requests`, `file handling`

## Task 4 - Basic Chatbot

**Goal:** A simple rule-based chatbot that responds to predefined user inputs.

**Features:**
- Responds to greetings like "hello", "hi", "hey"
- Answers "how are you", "what is your name", "help"
- Exits cleanly on "bye" or "quit"
- Handles unrecognized input gracefully

**Key Concepts:** `if-elif`, `functions`, `loops`, `input/output`
